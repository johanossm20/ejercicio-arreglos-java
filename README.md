# Ejercicio de Arreglos en Java

## Objetivo del proyecto

Desarrollar un programa en Java que permita trabajar con arreglos unidimensionales.
El programa solicita al usuario ingresar números en dos arreglos de tamaño 7, calcula la diferencia entre cada posición y almacena el resultado en un tercer arreglo.
Además, calcula el promedio de todos los números ingresados.

## Descripción del programa

El programa realiza las siguientes acciones:

* Define dos arreglos unidimensionales para almacenar 7 números ingresados por el usuario.
* Calcula la diferencia entre los valores correspondientes de los dos arreglos.
* Guarda el resultado en un tercer arreglo.
* Muestra todos los valores del tercer arreglo.
* Calcula y muestra el promedio de todos los números ingresados.

## Instrucciones de ejecución

1. Descargar o clonar el repositorio.

2. Compilar el archivo Java:

```
javac EjercicioArreglos.java
```

3. Ejecutar el programa:

```
java EjercicioArreglos
```

4. Ingresar los números solicitados por el programa.
